# DailyPlanner – Prototyp aplikacji mobilnej iOS

## Opis projektu

DailyPlanner to projekt koncepcyjny mobilnej aplikacji iOS do zarządzania codziennymi zadaniami.

Projekt obejmuje opracowanie interfejsu użytkownika (UI) oraz struktury nawigacji aplikacji. 
Na obecnym etapie wykonany został prototyp graficzny wszystkich widoków aplikacji.

Projekt został przygotowany w ramach przedmiotu:
**Aplikacje mobilne na iOS – Wykład + Lab01**

---

## Zakres projektu

W ramach zadania wykonano:

- Projekt interfejsu użytkownika w Figma
- Opracowanie struktury nawigacji między ekranami
- Przygotowanie wersji wizualnej (HTML + CSS) do podglądu projektu
- Dokumentację funkcjonalną aplikacji

Na tym etapie aplikacja stanowi prototyp wizualny (UI Design).

---

## Widoki aplikacji (5 ekranów)

1. **Home (Dashboard)**
   - Aktualna data
   - Lista zadań na dziś
   - Pasek postępu realizacji
   - Przycisk dodawania zadania

2. **All Tasks**
   - Lista wszystkich zadań
   - Filtrowanie: All / Completed / Pending

3. **Add Task**
   - Formularz dodawania zadania
   - Tytuł, opis, data, priorytet

4. **Task Detail**
   - Szczegóły zadania
   - Oznaczenie jako wykonane

5. **Settings**
   - Ustawienia powiadomień (symulacja)
   - Informacje o aplikacji
